## Data Structures and Algorithms

### Data Structures Used
| Data Structure | Purpose | Implementation |
|---------------|---------|----------------|
| LinkedList | Student records, attendance lists | Custom implementation |
| Stack | Undo functionality | Custom implementation |
| Queue | FIFO attendance processing | Custom implementation |
| BinarySearchTree | Name-based search | Custom implementation |
| AVLTree | Scalable student management | Custom implementation |
| PriorityQueue | Alert system | Custom implementation |
| HashTable | Fast ID lookups | Custom implementation |

### Algorithms Implemented
- **Searching**: Binary search tree traversal
- **Sorting**: Multiple sorting algorithms (by name, class, enrollment year)
- **Recursion**: Report generation, tree traversals
- **Tree Balancing**: AVL tree rotations
