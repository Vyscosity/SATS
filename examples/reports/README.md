# Team Reports

This folder contains the **fillable templates** for our team report and individual contribution reports.

## Files We Need to Fill In

1. **TEAM_REPORT.md** - Main team report (we fill this in)
2. **STUDENT1_CONTRIBUTION.md** - Student 1's contribution report (we fill this in)
3. **STUDENT2_CONTRIBUTION.md** - Student 2's contribution report (we fill this in)
4. **STUDENT3_CONTRIBUTION.md** - Student 3's contribution report (we fill this in)
5. **STUDENT4_CONTRIBUTION.md** - Student 4's contribution report (we fill this in)

## What We Need to Do

1. **Fill in all sections** marked with `[Insert...]` or `[Description]`
2. **Replace placeholder text** with our actual content
3. **Add our specific details**:
   - Names and student IDs
   - Code contributions
   - Challenges we faced
   - What we learned
   - Screenshots and diagrams

4. **Reference the examples** in the `examples/` folder if we need guidance on structure or formatting

## Tips

- Be specific: Include actual code examples, file names, and line numbers
- Be honest: Document both successes and challenges
- Be detailed: Explain our design decisions and reasoning
- Be collaborative: Acknowledge help from teammates
- Be reflective: Discuss what we learned and what we'd do differently

## Submission Checklist

- [ ] Team report completed
- [ ] All individual contribution reports completed
- [ ] All sections filled in
- [ ] Code examples included
- [ ] Screenshots/diagrams included (if applicable)
- [ ] References cited
- [ ] Proofread for grammar and spelling
- [ ] All team members reviewed

Good luck with our reports!
